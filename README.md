# Breast Cancer Classification Pipeline

This repository contains a clean, reproducible Machine Learning pipeline to predict patient survival status using the SEER Breast Cancer dataset. It includes data preprocessing, model training, and an inference pipeline using a Random Forest Classifier.

## 📁 Repository Structure
* **`data/`**: Directory for the input CSV files (ignored by git).
* **`src/`**: Modular Python scripts for running the pipeline steps sequentially.
